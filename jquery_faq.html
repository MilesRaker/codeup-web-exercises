FAQ about JQuery

How do I change css for all elements of a particular class?
Selector: $(".className") then use .css("styleName", "styleDefinition")
What is .html vs .text?
Differences are similar to innerText vs innerHtml
Does JQuery work with Bootstrap?
Yes. Bootstrap is not written with JQuery, although a developer may implement both Bootstrap and JQuery. Ana, thank you for this answer!
What is JQuery's .css() method?
.css() without arguments is a getter, .css("key","value") sets a new css rule for a JQuery object
What is a JQuery object?
An object created by $("selector") or jquery("selector"). Any JQuery methods may be used on these objects
Why do I want to use JQuery?
JQuery objects and methods are easier to use than document.getElementById() type methods. No need to set event listeners explicitly
How does addClass() work?
jqueryObject.addClass("className") adds a particular class to the selected element. That class can refer to a pre-existing style.
How about .removeClass()?
if a jqueryObject has "className", then remove that class from the element. Effectively removing a style.
...and how about .toggleClass()?
if jqueryObject does not have a class, add it. If it already has that class, remove it. Think: highlight when hovering.
Do I want to use JQuery?
Oh, heck yeah. It will be critical for all kinds of page interactivity.
